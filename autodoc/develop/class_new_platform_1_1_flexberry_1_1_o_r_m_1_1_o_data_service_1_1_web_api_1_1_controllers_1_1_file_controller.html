inline

Осуществляет загрузку файлов на сервер.

Файлы загружаются в файловую систему, в каталог UploadsDirectoryPath/{UploadedFileKey}, где UploadedFileGuid - Guid, идентифицирующий загруженный файл.

Возвращает
